                       
                    WELCOME TO BEARDOG!
                        
    Welcome to the Beardog Bookworm version
    allowing users to select and install packages
    based on want or need.
    
    To report any issues:
    https://www.forum.lilidog.org/

    To install Beardog, click on the green button
    in the system tray, or from the main menu -
    All Programs - System - Install Beardog. The 
    'root' password  for the session is 'live'.
    
    The debian installer is also an option should
    you prefer it. Reboot to access the menu.

    After installation, try the setup script in the
    welcome prompt or from the menu to see 
    other options you might like to have.
